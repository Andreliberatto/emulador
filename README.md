# DevOS

Sistema operacional no navegador feito com React + Vite.

## Recursos
- Desktop estilo sistema operacional
- Janelas arrastáveis
- Menu iniciar
- Barra de tarefas
- Navegador interno
- Explorador de arquivos com persistência local
- Editor de notas
- Terminal com comandos
- Player de música
- Launcher DOS com link configurável

## Como rodar

```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```
