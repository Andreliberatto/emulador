Simulador de Sistema Operacional Retro (Web)

Este projeto é um simulador de sistema operacional rodando no navegador, com foco em recriar a experiência de desktops antigos e permitir a execução de jogos MS-DOS diretamente na web.

A proposta é combinar uma interface inspirada em sistemas clássicos com tecnologias modernas, criando uma experiência interativa e nostálgica.

🚀 Funcionalidades
Interface desktop interativa
Janelas redimensionáveis, minimizáveis e organizáveis
Menu de aplicativos (estilo sistema operacional)
Game Center com execução de jogos DOS
Emulação de jogos diretamente no navegador
Persistência de dados via localStorage
🛠 Tecnologias utilizadas
React
Vite
Framer Motion
Lucide React
js-dos
🧠 Objetivo do projeto

O principal objetivo foi desenvolver uma aplicação que demonstrasse:

Construção de interfaces complexas com múltiplos estados
Experiência do usuário mais fluida com animações
Integração com bibliotecas externas (emulação)
Organização e escalabilidade de um projeto frontend
▶️ Como executar
npm install
npm run dev

Abra no navegador em:

http://localhost:5173
📌 Observações

O projeto foi desenvolvido com foco em portfólio, buscando explorar conceitos avançados de frontend de forma prática e criativa.
